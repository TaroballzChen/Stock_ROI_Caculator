# Stock_ROI_Caculator
A caculator of Taiwan stocks ROI
![Demo](https://i.imgur.com/VwvBWUX.png)

# Requirement
```commandline
PyQt5
pandas
lxml
beautifulsoup4
html5lib
```

# Setup
1. `pip install -r requirement`
2. Create database file with **csv** format
- Windows : `cd. > database.csv`
- Mac or Linux : `touch database.csv`

# Usage:
```commandline
python main.py
```
